Encouragement ऐनकरेजमैंट

सब ठीक हो जाएगा    -   Everything is going to be fine( एवरीथिंग इज़ गोइंग टू बी ओके )


बुरा मत सोच    -   Dont think negative( डोंट थिंक नैगेटिव )


अच्छा अच्छा सोच    -   Think positive( थिंक पॉज़ीटिव )


दुखी मत हो    -   Dont be sad( डोंट बी सैड )


चल हस    -   Smile( स्माइल )


आप कर सकते हैं    -   You can do this( यु कैन डु दिस )


आपके अंदर हुनर हैं    -   You are talented( यु आर टैलेंटेड )


मैं हु ना    -   I am here for you( आई ऐम हेयर फॉर यु )


आप अच्छा कर रहे हो    -   You are doing great( यु आर डूइंग ग्रेट )


लगे रहो    -   Keep it up( कीप इट अप )


आप इसी काम के लिए जन्मे थे    -   You were born for this
( यु वर बॉर्न फॉर दिस )
