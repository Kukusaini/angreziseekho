क्रोध करना

Getting angry गैटिंग ऐंग्री

चुप कर    -   Shut up( शट अप )


तेरे दांत तोड़ दूंगा    -   I will break your teeth( आई विल ब्रेक यौर टीथ )


मुझसे बात मत करो   -   Dont talk to me( डोंट टॉक टु मि )


मुझे अपना मुँह मत दिखाना    -   I dont want to see your face( आई डोंट वांट टु सी यौर फेस )


मुझे तुझसे नफरत हैं    -   I hate you( आई हेट यु )


मैं तुझे जान से मार दूंगा    -   I will kill you( आई विल किल यु )


मेरी नज़रो से दूर होजा    -   Get out of my sight( गैट आउट ऑफ माय साईट )


क्या हल्ला मचा रखा हैं    -   Stop making noise( स्टॉप मेकिंग नॉइज़ )


ये क्या बकवास हैं    -   What is this nonsense( वॉट इज़ दिस नॉनसैंस )


एक चाँटा पड़ेगा    -   I will slap you( आई विल स्लैप यु )
