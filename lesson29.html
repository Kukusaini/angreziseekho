मुलाकात

Meeting मीटिंग

जब आप किसी से पहली बार मिलते हैं तब बोला जाता हैं -


नमस्ते    -   Hello ( हैलो )

नमस्ते    -   Hi ( हाय )

जब वह इंसान आपसे उम्र में बड़े हो तब बोला जाता हैं -


नमस्ते महाशय    -   Hello sir( हैलो सर )

नमस्ते मोहतर्मा    -   Hello mam ( हैलो मैम )



अनजान व्यक्ति से पहली बार हाथ मिलाते हुए बोला जाता हैं -



आपसे मिलके बहुत ख़ुशी हुई    -   Nice to meet you ( नाइस टू मीट यू )

आपसे मिलके बहुत ख़ुशी हुई    -   Glad to meet you ( ग्लैड टू मीट यू )

आपसे मिलके बहुत ख़ुशी हुई    -   Pleased to meet you ( प्लीज़्ड टू मीट यू )



जान पहचान वाले व्यक्ति से मिलते वक्त बोला जाता हैं -



क्या हाल हैं    -   How are you ( हाओ आर यू )

आजकल क्या चल रहा हैं    -   Whats up ( वॉट्स अप )

और सब ठीक हैं    -   How is everything( हाओ इज़ ऐवरीथिंग )



किसी से बहुत दिन बाद मिलने पर बोला जाता हैं -


बड़े दिनों बाद दिखे आप    -   Long time no see( लॉन्ग टाइम नो सी)



किसी से विदा लेते समय बोला जाता हैं -



चलिए कल मिलते हैं    -   See you tomorrow( सी यू टुमॉरो )

चलिए जल्दी मिलते हैं    -   See you soon ( सी यू सून )

अलविदा    -   Goodbye / Bye ( गुडबाय / बाय )

अच्छा ठीक हैं फिर    -   Okay Goodbye ( ओके गुडबाय )



कृपया इस बात का ध्यान दें रात के समय विदा लेते समय
गुडबाय की जगह गुडनाइट बोला जाता हैं -



अलविदा    -   Goodnight ( गुडनाइट )

अच्छा ठीक हैं फिर    -   Okay Goodnight ( ओके गुडनाइट )
