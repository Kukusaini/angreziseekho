क्रिकेट मैच

Cricket Match

कौनसी टीम हैं    -   Which team is it( विच टीम इज़ इट )


क्या स्कोर चल रहा हैं    -   What is the score( वॉट इस इज़ स्कोर )


बल्लेबाज़ कौन हैं    -   Who is batting( हु इज़ बैटिंग )


गेंदबाज़ कौन हैं    -   Who is bowling( हु इज़ बौलिंग )


वाह क्या गेंद फेंकी हैं    -   Good ball( गुड बॉल )


वाह क्या मारा हैं    -   What a shot( वॉट अ शॉट )


सचिन मेरा पसंदीदा हैं    -   Sachin is my favourite( सचिन इज़ माय फेवरेट )


बल्लेबाज़    -   Batsman( बैट्समैन )


गेंदबाज़    -   Bowler( बोलर )


छक्के चौको की बरसात हो रही हैं    -   Its raining fours and sixes
( इट्स रेनिंग फ़ोर्ज़ ऐंड सिक्सिस )


युवराज कमाल का बल्लेबाज़ हैं    -   Yuvraj is a great batsman
( युवराज इज़ अ ग्रेट बैट्समैन )


ज़हीर अच्छा गेंदबाज़ हैं    -   Zaheer is a good bowler( ज़हीर इज़ अ गुड बोलर )


बेकार फील्डिंग    -   Poor fielding( पुअर फील्डिंग )


बढ़िया बल्लेबाज़ी    -   Good batting( गुड बैटिंग )


बढ़िया गेंदबाज़ी    -   Good bowling( गुड बौलिंग )


आसमानी छक्का    -   Huge six( ह्यूज सिक्स )


बॉल गोली की रफ्तार से गयी    -   Ball went like a bullet( बॉल वैंट लाइक अ बुलेट )


बल्ले की रफ्तार कम थी    -   Timing was poor( टाइमिंग वाज़ पुअर )


चौका लगा    -   Its a four( इट्स अ फोर )


छक्का लगा    -   Its a six( इट्स अ सिक्स )


अंपायर ने आउट दिया हैं    -   Umpire has given out( अंपायर हैज़ गिवन आउट )
