Past पास्ट

वो भी क्या दिन थे    -   Those were the days( डोज़ वर दा डेज़ )


ये तब की बात हैं    -   Back in the past( बैक इन दा पास्ट )


सन 1990 में    -   In 1990( इन 1990 )


90 के दशक में    -   In the ninetees( इन दा नाइंटीस )


प्राचीन काल में    -   In ancient times( इन एंशियंट टाइम्स )


बहुत समय पहले    -   A long time ago( अ लॉन्ग टाइम अगो )


कुछ समय पहले    -   A while earlier( अ वाईल अर्लियर )


दस साल पहले    -   10 years ago( 10 इयर्ज़ अगो )


क्या तुम्हे याद हैं    -   Do you remember( डु यु रिमैम्बर )


उस दिन    -   That day( दैट डे )


कुछ मिनट पहले    -   Few minutes ago( फ्यू मिनिट्स अगो )


कुछ घंटे पहले    -   Few hours ago( फ्यू आर्ज़ अगो )


कुछ दिन पहले    -   Few days ago( फ्यू डेज़ अगो )


कुछ साल पहले    -   Few years ago( फ्यू इयर्ज़ अगो )


कुछ पल पहले    -   Few moments ago( फ्यू मोमैन्ट्स अगो )


बीता हुआ कल    -   Yesterday( यैसटरडे )


पिछला महीना    -   Last month( लास्ट मंथ )


पिछला साल    -   Last year( लास्ट ईयर )


पिछला दशक    -   Lst decade( लास्ट डैकेड )


कुछ दशक पहले    -   Few decades( फ्यू डैकेड्ज़ अगो )


एक मिनट पहले    -   A minute ago( अ मिनिट अगो )


एक घंटे पहले    -   An hour ago( ऐन आर अगो )


से पहले    -   before( बिफोर )


विभाजन से पहले    -   before partition( बिफोर पार्टीशन )
