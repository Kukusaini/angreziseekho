Personality पर्सनैलिटी

बुरा इंसान    -   Bad person( बैड पर्सन )


अच्छा इंसान    -   Good person( गुड़ पर्सन )


वह अच्छा इंसान हैं    -   He is a good person( ही इज़ अ गुड पर्सन )


वह बुरी इंसान हैं    -   She is a good person( शी इज़ अ बैड पर्सन )


खुश    -   Happy( हैप्पी )


ख़ुशी    -   Happiness( हैप्पीनैस )


वह हमेशा खुश रहता हैं    -   He is always happy( ही इज़ ऑल्वेज़ हैप्पी )


शर्मिला    -   Shy( शाए )


वह शर्मिला हैं    -   He is shy( ही इज़ शाए )


सामाजिक    -   Social( सोश्यल )


इंसान सामाजिक प्राणी हैं    -   Human is a social being( ह्यूमन इज़ अ सोश्यल बीईंग )


घुल-मिल के रहना वाला    -   Extrovert( एक्सट्रोवर्ट )


अकेले-अकेले रहने वाला    -   Introvert( इंट्रोवर्ट )


तेज़ दिमाग वाला    -   Sharp-minded( शार्प-माइन्डिड )


बुद्धिमान    -   Intelligent( इंटैलिजैंट )


उसका दिमाग तेज़ हैं    -   He is intelligent( ही इज़ शार्प-माइन्डिड )


चालाक    -   Cunning( कंनिंग )


विनम्र    -   Polite( पोलाइट )


वह विनम्र हैं    -   He is polite( ही इज़ पोलाइट )


बद्तमीज़    -   Rude( रूड )


मज़ाकिया    -   Funny( फनी )
