Singular noun ( सिंगुलर नाउन )


को हिंदी मे एकवचन कहते हैं।

एकवचन मतलब किसी भी चीज़ का संख्या मे एक होना।


उदाहरण


Boy ( बॉय ) - लड़का (संख्या में एक )

One piece (वन पीस) - एक टुकड़ा / टुकड़ा

Apple ( एप्पल ) - सेब (संख्या में एक )

Forest ( फॉरेस्ट ) - जंगल



चाहे आप बोले Apple ( एप्पल ) या बोले One Apple ( एक एप्पल )

इन दोनों का मतलब हैं - एक सेब

यह नियम हर एक Singular ( सिंगुलर ) शब्द पर लागू होता हैं।



Plural noun ( प्लुरल नाउन )


को हिंदी मे बहुवचन कहते हैं।

बहुवचन मतलब किसी भी चीज़ का संख्या मे एक से ज्यादा होना।


उदाहरण


Some Boys ( सम बॉयज ) - कुछ लड़के

Six pieces (सिक्स पीसिज़ ) - छ: टुकड़े

Two Apple ( दो एप्पल ) - दो सेब

Many places( मैनी प्लेसेस ) - अनेक जगह



अब हम सीखेंगे


एकवचन( Singular ) को बहुवचन( Plural ) मे बदलना।



नियम नंबर 1



आमतौर पर शब्दों( Words ) के आगे   s   लगाने पर सिंगुलर से प्लुरल बनाया जाता है।


उदाहरण


लड़का  Boy( बॉय )   +  s   =   Boys( बॉयज़ ) - लड़के


लड़की  Girl( गर्ल )  +  s   =   Girls( गर्ल्स ) - लड़किया


गेंद   Ball( बॉल )  +  s   =   Balls( बॉल्स ) - गेंदे


कुर्सी  Chair( चेयर )  +  s   =   Chairs( चेयर्स ) - कुर्सियां



नियम नंबर 2



जिन शब्दों( Words ) का अंत x , sh , ch , s , z से होता है

उन शब्दों( Words ) के आगे   es   लगाने पर सिंगुलर से प्लुरल बनाया जाता है।


उदाहरण


Brush( ब्रश )   +  es   =   Brushes( ब्रशीज़ )


Lunch( लंच )  +  es   =   Lunches( लंचिज़ )


Fox( फॉक्स )  +  es   =   Foxes( फोक्सीज़ )


Success( सक्सेस )  +  es   =   Successes( सक्सेसइज़ )


Buzz( बज़्ज़ )  +  es   =   Buzzes( बज़्ज़इज़ )



नियम नंबर 3



जिन शब्दों(Word) के अंत में y आता है। उन शब्दों मे y से पहले वाला

शब्द देखा जाता है।

- अगर y से पहले कोई Vowel ( a , e , i , o , u ) आता है

तो वर्ड के आगे s लगा दिया जाता हैं।


उदाहरण


Boy( बॉय )        =       Boys( बॉयज़ )


Monkey( मंकी )       =       Monkeys( मंकीज़ )



- अगर y से पहले कोई Vowel ( a , e , i , o , b ) नहीं आता है

तो शब्द (Word) में से y को हटाकर ies लगाया जाता हैं।

उदाहरण


Baby( बेबी )       =       Babies( बेबीज़ )


Family( फैमिली )       =       Families( फैमिलीज़ )


Duty( ड्यूटी )       =       Duties( ड्यूटीज़ )



नियम नंबर 4



जिन शब्दों( Words ) का अंत f , ff , fe से होता है

उन शब्दों( Words )में से ( f , ff , fe ) को हटाकर ves लगाया जाता हैं।


उदाहरण


Wolf( वुल्फ )       =       Wolves( वूल्व्स )

कुछ ख़ास शब्द जिनके प्लुरल ( बहुवचन )

नियमो का पालन नहीं करते परन्तु आम ज़िन्दगी में इस्तेमाल होते हैं और

आपको पता होना चाहिए।


Man (मैन)       =       Men ( मेन )


Child (चाइल्ड )       =       Children (चिल्ड्रन )


Tooth ( टूथ )       =       Teeth ( टीथ )
