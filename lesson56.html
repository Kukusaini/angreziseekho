Journey जर्नी

चलो जयपुर चलते हैं    -   Lets go to Jaipur( लैट्स गो टू जयपुर )


चार टिकट बुक करवा दो    -   Book four tickets( बुक फोर टिकिट्स )


मुझे खिड़की वाली सीट चाहिए    -   I want window seat( आई वॉन्ट विंडो सीट )


यात्रा कितने घंटे की हैं    -   How long is the journey( हाओ लॉन्ग इज़ दा जर्नी )


मज़ा आने वाला हैं    -   Its going to be fun( इट्स गोइंग टू बी फन )


यहाँ मशहूर जगह कौनसी हैं   -   What are famous places here
( वॉट आर फेमस प्लेसिस हेयर )


यहाँ मशहूर पकवान कौनसे हैं   -   What are famous dishes here
( वॉट आर फेमस डिशिस हेयर )


पुष्कर क्यों मशहूर हैं    -   What is Pushkar famous for( वॉट इज़ पुष्कर फेमस फॉर )


प्रस्थान का समय क्या हैं    -   When is the departure( वैन इज़ दा डिपार्चर )


कितने बजे पहुँचेंगे    -   When is the arrival( वैन इज़ दा अराइवल )


सब तैयार हैं क्या    -   Is everyone ready( इज़ ऐवरिवन रैडी )


जल्दी करो    -   Hurry up( हरि अप )


एक अख़बार खरीद लो    -   Buy a newspaper( बाए अ न्यूज़पेपर )


एक पानी की बोतल ले लो    -   Take a water bottle( टेक अ वॉटर बॉटल )


शुभ यात्रा    -   Happy journey( हैप्पी जर्नी )
